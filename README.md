# 🍽️ MAREN Restaurant

A modern fine-dining restaurant website designed to deliver a luxurious digital experience through elegant design, sophisticated storytelling, and a fully responsive interface.

MAREN is a fictional Michelin-inspired restaurant concept created as a front-end development project focused on premium branding, menu presentation, reservation functionality, and refined user experience.

## Overview

This project was built to simulate the online presence of a high-end fine dining restaurant. The design emphasizes visual hierarchy, elegant typography, spacious layouts, and storytelling to create a premium and memorable browsing experience.

The website showcases the restaurant's story, seasonal menu, culinary team, and reservation system while maintaining responsiveness across all devices.

## Features

* Fully responsive design
* Premium restaurant-inspired user interface
* Hero section with strong visual storytelling
* Restaurant story and brand identity
* Seasonal menu showcase
* Chef profiles and team introduction
* Reservation form
* Smooth navigation experience
* Mobile-friendly layout

## Sections

### Hero Section

A bold introduction highlighting the restaurant's identity, location, and fine dining experience.

### Our Story

Presents the restaurant's journey, philosophy, and commitment to culinary excellence.

### Seasonal Menu

Showcases selected dishes from the restaurant's tasting menu with descriptions and pricing.

### Meet Our Chefs

Introduces the culinary team behind the restaurant and adds personality to the brand.

### Reservations

Allows guests to reserve a table through a clean and intuitive booking interface.

### Footer

Contains navigation links, restaurant information, and branding.
